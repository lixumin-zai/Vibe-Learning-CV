# Mind Map Tool

Use the whiteboard below to create mind maps, flowcharts, or diagrams for your learning path.

<div className="my-8 not-prose">

</div>

## How to use
- Click the **Open Mind Map Whiteboard** button to launch the editor.
- **Double click** to add text.
- Use the **arrow tool** to connect nodes.
- **Export** your diagram as PNG/SVG to save it locally.
