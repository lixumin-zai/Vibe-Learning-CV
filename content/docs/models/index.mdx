# Common Models

A catalog of industry-standard models for various vision tasks.

## Object Detection
- **YOLO (You Only Look Once)**: Real-time object detection.
- **R-CNN Family**: Fast R-CNN, Faster R-CNN, Mask R-CNN.
- **SSD (Single Shot Detector)**: Balancing speed and accuracy.

## Segmentation
- **U-Net**: Biomedical image segmentation.
- **DeepLab**: Semantic image segmentation with deep convolutional nets.

## Classification
- **ResNet**: Deep residual networks.
- **MobileNet**: Efficient models for mobile applications.
- **ViT**: Vision Transformers.
